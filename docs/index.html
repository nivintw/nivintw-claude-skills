Claude Code · plugin marketplace

Tooling you install and invoke by typing.

Tyler Nivin's Claude Code plugins — a scriptable terminal-recording toolkit and a human + AI development workflow. Add the marketplace, install a plugin, then just ask Claude Code.

$/plugin marketplace add nivintw/nivintw-claude-skills

Plugins

Three plugins — twelve commands and a worktree-safety hook. Search above, or browse below.

castify

documentation

Scriptable asciinema recordings. Script every keystroke and pause instead of performing a session live, so casts are reproducible and deterministic — and so you can record interactive TUIs like fzf, less, and vim at all.

/castify:record-terminal-casts

dev-kit

workflow

A human + AI teaming development workflow. Take a change from idea to a reviewed PR, track the work as issues, pick what to do next, and keep your local clone and docs tidy — with the human in control at the ends.

/dev-kit:ship /dev-kit:review-pr /dev-kit:generate-docs

worktree-guard

workflow

A safety net for git-worktree work. One PreToolUse hook that blocks an accidental write to the parent checkout while you're in a worktree — editing main's copy instead of the worktree's. Inert outside a worktree; fail-open on any error.

worktree-guard ↗

Install

From the marketplace, or from a local clone.

# add the marketplace /plugin marketplace add nivintw/nivintw-claude-skills # install a plugin /plugin install castify@nivintw-claude-skills /plugin install dev-kit@nivintw-claude-skills /plugin install worktree-guard@nivintw-claude-skills

From a local clone, point the marketplace at the directory instead: /plugin marketplace add ~/workspace/nivintw-claude-skills. Then ask Claude Code for what you want — the skills activate on their own.
